# Temperature Test Örnekleri

## Aynı Soruya Farklı Temperature'larda Cevaplar

**Soru:** "İstanbul'u anlat"

### Temperature 0.0 (Robotik)
"İstanbul, Türkiye'nin en kalabalık şehri ve eski başkentidir. 
Boğaziçi'nin iki yakasına kurulmuştur."

### Temperature 0.7 (Dengeli) ✅
"İstanbul müthiş bir şehir! Hem Avrupa hem Asya'da, Boğaz'ın iki yakasında. 
Tarihi dokusu, modern hayatla harmanlanmış. Mutlaka görmelisin!"

### Temperature 1.5 (Çok Yaratıcı)
"Ah İstanbul! Zamanın aynasında dans eden bir şehir... 
Minareler gökyüzünü öperken, martılar özgürlüğün türküsünü söyler. 
Boğaz'ın maviliğinde Asya ile Avrupa el ele..."

## Kod Yazımı Örneği

**Soru:** "Python'da merhaba yazdır"

### Temperature 0.0 (En iyi kod için)
```python
print("Merhaba")
```

### Temperature 0.7
```python
# Merhaba mesajı yazdırma
print("Merhaba")
```

### Temperature 1.5 (Gereksiz yaratıcı)
```python
# Dünyaya selam ver!
mesaj = "Merhaba"
print(f"🌍 {mesaj}!")
```
